# 常见的图片格式

- BMP: 原像素存储图片。常见 Mode: `1`, `L`, `P`, `RGB`。
- PNG: 无损压缩图片。常见 Mode: `1`, `L`, `LA`, `P`, `RGB`, `RGBA`。
- JPEG / JPG: 有损压缩图片。常见 Mode: `L`, `RGB`, `CMYK`。
- HEIF / HEIC: 有损压缩图片，但是压缩效率更高。常见 Mode: `L`, `LA`, `RGB`, `RGBA`。

# 图片包含的信息

- MIME type: 用于识别图片格式的元信息字段。
- Mode: 用于表示解码后单个像素由哪些颜色或透明度通道组成，以及通道如何表达。
    - 1: 黑白二值图。
    - L: 灰度图。
    - P: 调色板索引模式，每个像素是一个索引值，其目标位置其实还是 RGB。
    - LA: 灰度加透明度。
    - RGB: 彩色图。
    - RGBA: 彩色透明图。
    - CMYK: 印刷颜色模式，Cyan / Magenta / Yellow / Key 是常见的几种印刷油墨颜色。可以由 RGB 推算近似值，但是由于色域不同无法精确推算。
- Metadata: 文件格式、显示、编码相关的各类图片信息。
- EXIF: 图片的拍摄信息，日期、地点、设备、镜头参数等附带信息。
